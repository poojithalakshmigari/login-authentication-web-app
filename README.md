
# Login Authentication Web App

A simple login authentication web application developed using HTML, CSS, and JavaScript.

## Features
- User login interface
- Form validation
- Username and password authentication
- Responsive UI design

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Files
- index.html – Login page structure
- style.css – Styling and layout
- script.js – Login validation logic
